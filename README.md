# Betfair

A library for the Betfair Exchange API. Not yet usable, still in early development stage.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'betfair', github: 'mikecmpbll/betfair'
```

And then execute:

    $ bundle
